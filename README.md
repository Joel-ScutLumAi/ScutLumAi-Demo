# ScutLumAi — Structural Signal Demo

**Concept Prototype · MVP-1.0**

ScutLumAi is a structural signal assessment infrastructure designed to extract, classify, and render observable relationships within bounded digital content.

This demo shows the constitutional engine's signal detection on a curated subset of the full taxonomy — applied to financial publishing copy.

---

## What this is

A sentence-level structural analysis tool that detects:

- **Confidence signals** — certainty expressions within a bounded unit
- **Support signals** — evidentiary anchors within the same unit
- **Structural correspondence** — the relationship between the two
- **Signal modifiers** — scope, amplification, and hedging markers

## What this is not

- Not a writing assistant
- Not a misinformation detector
- Not a risk rating tool
- Not a recommendation engine

The engine describes structure. Interpretation remains with the human editor.

---

## Stack

- Next.js 14
- Deployed on Vercel (Hobby)
- No database. No API calls. Fully static demo.

---

## Constitutional Constraint

> The engine does not determine truth. It does not assign risk ratings. It does not produce probabilistic conclusions. Its sole mandate is signal visibility.
